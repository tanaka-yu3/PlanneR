■ 旅行者向けマッチングサイト
### サイトテーマ
■ オススメを教えよう、知ろう

■ 私自身も旅行が好きで、現地のオススメスポットを現地の方がユーザーのニーズで案内できたら可能性があると感じた  
■ 5Gの時代になり個人での発信、稼ぐことへシフトするため  
■ 物消費　→　事消費へと消費の対象の変化
